COMO UTILIZAR SUBLIME TEXT

Mi Página Web

Tanto si escribes código HTML, como si creas código fuente o escribes textos sencillos, Sublime Text es uno de los editores de texto más valorados. Esto se debe a su compatibilidad multiplataforma y a sus diversas posibilidades de edición de texto para el desarrollo de software. Los editores de texto son útiles para escribir archivos de texto plano y para editar, compartir y publicar el código fuente de programas y páginas web. Uno de los editores de texto más conocidos es Sublime Text, desarrollado en 2007, entre otros, por el desarrollador de software de Google Jon Skinner en C++ y Python. En un principio, Sublime Text funcionaba solo en Windows, si bien ahora es compatible con otras plataformas y se puede ampliar de forma flexible. Durante mucho tiempo, Sublime Text 3 fue la última versión del editor hasta que Sublime Text 4 la sustituyó en julio de 2022.

¿Por qué es tan especial Sublime Text?

La gran ventaja de Sublime Text reside en su amplia gama de funciones y el bajo consumo de recursos. Al ser un editor todo en uno, Sublime Text no es solo un editor de texto sin formato, sino que puede utilizarse para programar código complejo. Las principales ventajas de Sublime Text son su facilidad de uso y la flexibilidad para ampliarlo. Las características más importantes de Sublime Text son:

  • Resaltado de sintaxis
  • Acabado del código
  • Uso de macros
  • Edición multilínea
  • Edición paralela de datos en varias ventanas
  • Integración del compilador
  • Ajuste flexible de la configuración
  • Función de búsqueda en diálogos, archivos, documentos y líneas
  • Accesos directos
  • Edición modal
  • API de Python
  • Selección de varias pestañas
  • Modo oscuro opcional
  • API de plugins para extensiones

    • Además, el editor es compatible con los sistemas operativos principales. Puedes descargar Sublime Text para Mac, Windows y Linux.

    ¿Cómo se descarga Sublime Text?

    Sublime Text puede descargarse e instalarse como versión gratuita desde la página web oficial de Sublimetext.com o a través de un portal de descargas seguro y de buena reputación de proveedores verificados como Google Play. Puedes encontrar versiones anteriores (Sublime Text 3 y 2) en la parte inferior de la página de descargas de la web oficial o en “Other Downloads”.

    ¿Está disponible Sublime Text de forma gratuita?

    Sublime puede utilizarse sin coste alguno durante un periodo de tiempo ilimitado. Sin embargo, la versión gratuita suele mostrar ventanas emergentes que recuerdan a los usuarios la existencia de una licencia de pago. Cada licencia es válida para un usuario y puede utilizarse en tantos dispositivos como se desee. Cuesta 99 dólares estadounidenses al año (a fecha diciembre de 2022). Obtener una licencia presenta las siguientes ventajas:

  • Desaparecen las ventanas emergentes
  • Acceso a las versiones beta de Sublime Text
  • Mensajes actualizados en la app

    • ¿Cómo se utiliza Sublime Text?

    Utilizar Sublime Text de una forma o de otra depende del modo de empleo que le quieras dar. El editor es adecuado tanto para escribir texto plano como código complejo. Estos son algunos de los escenarios de aplicación más frecuentes de Sublime Text.

    Ajustar la configuración

    Los ajustes del editor se basan en texto y se encuentran en “Settings” guardados en formato JSON. Los valores clave de la configuración de Sublime Text se ajustan en dos matrices. La matriz de la izquierda contiene los valores por defecto; la de la derecha tiene los valores y ajustes personalizados. Para ajustar la configuración, debes hacer los cambios en la matriz JSON de la derecha. Esto te permite restablecer los valores por defecto de la izquierda más adelante si lo necesitas. También es una buena idea utilizar una herramienta de validación como JSONLint cuando hagas cambios y reformatees.

    Gestión de proyectos y archivos

    Desde borradores de código fuente hasta texto plano para blogs y revistas, los archivos en Sublime Text se almacenan y organizan en sus respectivas carpetas de proyecto. Para ello se dispone de una interfaz gráfica de gestión de proyectos. Es útil para crear nuevos archivos o renombrar, abrir, editar o eliminar los existentes. Los nuevos proyectos se crean de forma automática cuando creas nuevos archivos.

    Utilizar e integrar las extensiones

    Una de las características más prácticas de Sublime Text es su sencilla capacidad de ampliación mediante plugins y paquetes. Algunos de los plugins más útiles son:

  • Sidebar Enhancements: comandos y ajustes adicionales para el menú contextual de la barra lateral
  • SideBarGit: todos los comandos de Git para usar Github aparecen en una lista en la barra lateral
  • HTML5: finalización completa del HTML
  • SublimeCodeIntel: completar el código de los lenguajes de programación web más importantes y comunes de Internet
  • SublimeLinter: proporciona un framework para instalar los linters deseados para depurar los códigos de programación
  • Formatter: proporciona reglas de formato para Python, HTML, CSS o JS, así como formato automático y estandarización del código fuente.

    • ¿Quién puede sacar el máximo provecho del editor?

    Sublime Text es idóneo para los usuarios que se inician y son nuevos en el aprendizaje de programación y en la creación de código fuente, pero también para programadores, redactores y editores profesionales. A continuación, se muestran algunos aspectos del editor por cada grupo de usuarios.

    Para desarrolladores y programadores

  • Ampliable de forma flexible mediante la extensa gama de plugins del gestor de paquetes

  • Edición en paralelo de varias líneas (en ventanas múltiples o compartidas)

  • Admite sistemas de compilación sin necesidad de un terminal externo

  • Personaliza la interfaz de usuario con temas individuales y esquemas de color para resaltar pestañas, menús y sintaxis

  • Inserta fragmentos de texto como bloques de texto y busca y reemplaza con expresiones regulares elementos específicos de texto y código

  • Práctico para crear programas y editar páginas web, por ejemplo, en combinación con un flat-file CMS

    • Para redactores, blogueros y editores

  • Gestión de archivos para organizar los textos por proyecto
  • Busca y reemplaza archivos y carpetas o elementos de texto en las carpetas seleccionadas
  • Ampliación opcional de la sintaxis markdown

    • Conclusión: muchas ventajas, un pequeño inconveniente

    Desde la funcionalidad multiplataforma hasta la configuración personalizada y la posibilidad de ampliación flexible, Sublime Text ofrece todo lo que necesitan los desarrolladores y redactores. Se puede utilizar de forma gratuita y, aunque los mensajes emergentes animando a una suscripción pueden resultar molestos, es un pequeño peaje que pagar por usar uno de los mejores editores de texto gratuitos en el mercado. Sin embargo, hay un pequeño inconveniente: el menú de configuración basado en texto y los sofisticados atajos presentan una curva de aprendizaje más difícil para los principiantes.
